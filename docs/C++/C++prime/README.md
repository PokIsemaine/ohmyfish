## 书本介绍

C++ 学习的**经典著作**

这本书**很厚**，至少比我的笔记本电脑厚多了，因此书中内容很丰富，可以说是面面俱到了。

这本书**不适合入门**，对于没有其他编程语言学习经历的人来说有较大难度，很多基础的东西默认你已经懂了

书本的**代码并不会完整给出**，需要自己通过上下文和写出完整的代码
