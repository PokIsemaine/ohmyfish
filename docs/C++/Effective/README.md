## 注意事项

* 由于精力有限，Effective 系列的学习笔记只会摘录条款和重点，并不会细纠为什么要这么做（实际上，如果你仔细阅读过《C++ Primer》应该是知道大概原因的）
* 有些条款会随着`C++`标准的变化而失效，请结合实际需要来考虑是否采纳相关条款
* 条款全部背下来也没啥用，要成为习惯，自然而然写出来。
